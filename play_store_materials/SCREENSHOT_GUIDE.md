## 📸 How to Take Screenshots

### Method 1: Chrome DevTools (Best for Web)
1. Open Chrome DevTools: Press F12
2. Click "Toggle Device Toolbar" icon (phone/tablet icon) or press Ctrl+Shift+M
3. Select device: "iPhone 12 Pro" or similar (390 x 844)
4. Navigate through your app
5. Take screenshot: Windows Snipping Tool (Win+Shift+S) or Chrome screenshot feature
6. Save each screenshot with descriptive name

### Method 2: Windows Snipping Tool
1. Resize browser window to mobile size (375px wide)
2. Press Win+Shift+S
3. Select area to capture
4. Screenshot copies to clipboard
5. Paste into Paint and save as PNG

### Method 3: Chrome Built-in Screenshot
1. Open DevTools (F12)
2. Enable device mode (Ctrl+Shift+M)
3. Press Ctrl+Shift+P
4. Type "screenshot"
5. Choose "Capture screenshot" (full size) or "Capture node screenshot"

## 📐 Screenshot Requirements

**Current Size**: Screenshots will likely be your browser window size
**Required Size**: 1080 x 1920 pixels (9:16 aspect ratio)

### Resize After Capture

**Option 1: Online Tool**
- Go to: https://www.befunky.com/create/resize-image/
- Upload your screenshot
- Resize to: 1080 x 1920
- Download

**Option 2: Canva**
- Upload screenshot to Canva
- Resize canvas to 1080 x 1920
- Position image to fit
- Download

**Option 3: ImageMagick (if installed)**
```powershell
magick convert input.png -resize 1080x1920 output.png
```

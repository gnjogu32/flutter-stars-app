# How to Use Direct Messaging in Starpage

## Quick Start Guide

### Access Messages

**Option 1: From Bottom Navigation**
1. Open Starpage app
2. Tap the **Messages** icon in the bottom navigation bar
3. See all your conversations

**Option 2: From User Profile**
1. Find a user's profile (via Search or Followers)
2. Tap the **Message** button below their name
3. ChatScreen opens with that user

## Starting a Conversation

### Method 1: Message Button on Profile
1. Navigate to any user's profile
2. Scroll down to action buttons
3. Click **Message** button (next to Follow button)
4. ChatScreen opens automatically

### Method 2: Existing Conversation
1. Go to **Messages** tab
2. Tap any conversation from the list
3. Opens chat with that user

### Method 3: Search from Messages Tab
1. Go to **Messages** tab
2. Use search box at top
3. Type user's name
4. Filtered conversations appear
5. Tap to open

## Sending Messages

### How to Send a Message
1. Open conversation with a user
2. Tap text input field at bottom
3. Type your message
4. Click **send button** (blue circle with arrow)
5. Message appears in conversation immediately
6. Other user receives notification

### Message Bubble Colors
- **Blue bubbles** = Your messages (right side)
- **Gray bubbles** = Other user's messages (left side)

### Message Status
- **Checkmark icon** = Other user has read your message
- **No checkmark** = Message sent but not read yet

## Viewing Your Conversations

### Messages List View
Shows all your active conversations sorted by most recent

**For Each Conversation:**
- Profile picture of other user
- Their display name
- Preview of last message sent
- Time since last message (e.g., "2h ago")
- **Blue badge** showing unread message count (if any)

### Empty State
If you haven't messaged anyone yet:
- Message icon displayed
- "No conversations yet" text
- "Start a conversation with someone..." tip

## Searching Conversations

### Search Feature
1. Go to **Messages** tab
2. Look for search box at top
3. Type the other user's name
4. List filters automatically
5. Conversations matching that name appear

### Clearing Search
- Clear the text field to show all conversations again

## Troubleshooting

### Message Not Sending?
- Check internet connection
- Verify both users exist
- Wait a moment for connection to Firebase

### Can't Find User?
- Use search feature to find them
- Make sure you're spelling name correctly
- Navigate to their profile and use Message button there

### Notification Not Appearing?
- Check Notifications tab
- Notification only appears if you're not already in that chat
- Refresh app if needed

### Messages Disappearing?
- This shouldn't happen - all messages are in Firestore
- Try refreshing the conversation
- Messages are permanent once sent

## Privacy & Safety

### Who Can Message Me?
- Any Starpage user can start a conversation with you
- No restrictions or friend requirements

### Message Privacy
- Messages only visible to the two participants
- Other users cannot see your conversations
- All messages stored securely in Firebase

## FAQs

**Q: Can I send images in messages?**
A: Not yet, but this feature is planned. Currently text messages only.

**Q: Can I edit a message I sent?**
A: Not yet, but you can delete and resend.

**Q: How many messages can I send?**
A: Unlimited! No message limit.

**Q: Do messages disappear after a while?**
A: No, all messages are permanent and stored in Firestore.

**Q: Can I see old messages?**
A: Yes, scroll up in the conversation to see older messages.

**Q: What if I delete a conversation?**
A: The conversation and all its messages are deleted. They cannot be recovered.

**Q: Do I get notified for every message?**
A: Yes, but only if you're not currently in that chat window.

**Q: Can two users message at the same time?**
A: Yes! Real-time updates mean you'll see messages as they're typed.

## Getting Help

If you experience issues:
1. Check your internet connection
2. Try refreshing the app (R key in terminal during development)
3. Log out and log back in
4. Clear app cache and reinstall

For bugs, contact the development team with details about what went wrong.

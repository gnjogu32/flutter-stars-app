## 3. App Icon (Required)

### Specifications
```
Dimensions: 512x512 pixels
Format: PNG with transparency
Safe area: 90x90 to 422x422 pixels (center)
Style: Flat design recommended
File size: < 1 MB
```

### Design Guidelines
- âœ… Must be recognizable at small sizes (32x32 px)
- âœ… Should represent app purpose
- âœ… Use brand colors
- âœ… Avoid complex details
- âŒ Don't include app name in icon
- âŒ Avoid abstract or unclear imagery

### Recommended Design Elements
For Starpage:
- Star icon (represents "Star" in name)
- Page or document element
- Vibrant colors (blue, purple, orange suggested)
- Modern, flat design style

**Action**: Create or source a 512x512 PNG icon
- Can use: Figma, Canva, Adobe XD, or hire designer
- Budget: $0 (DIY) to $200 (professional)

### Specifications
```

Format: PNG or JPEG
```

## 6. Privacy Policy (Required)

### Must Include
- [ ] Data collection practices
- [ ] How user data is used
- [ ] Third-party services (Firebase, Google Analytics)
- [ ] User authentication methods
- [ ] Contact information
- [ ] Data retention policies
- [ ] User rights (delete, export, etc.)

### Privacy Policy Template
```markdown
# Starpage Privacy Policy

Last Updated: January 9, 2026

## 1. INTRODUCTION
Starpage ("we", "us", "our", or "Company") operates this application.
This page informs you of our policies regarding data collection and use.

## 2. INFORMATION COLLECTION AND USE

### User Account Information
- Email address
- Username
- Profile information
- Password (hashed for security)

### User-Generated Content
- Posts and comments
- Direct messages
- Profile pictures
- Following/follower relationships

### Automatically Collected Information
- Device information (model, OS)
- IP address and location
- App usage statistics
- Crash reports (optional)

## 3. THIRD-PARTY SERVICES

### Firebase (Google)
- Authentication
- Database storage
- Cloud Firestore
- See: https://firebase.google.com/support/privacy

### Analytics
- App usage tracking
- Performance monitoring
- Error logging

## 4. DATA SECURITY
We implement security measures to protect user data:
- Encryption in transit (HTTPS/TLS)
- Secure authentication
- Regular security audits

## 5. USER RIGHTS
Users can:
- Update profile information
- Delete their account
- Download their data
- Opt-out of analytics

To exercise rights: privacy@starpage.org

## 6. CHILDREN'S PRIVACY
[If applicable to your app]
- This app is intended for users 13+
- We don't knowingly collect data from children under 13
- Parents can contact us to request data deletion

## 7. CONTACT US
Questions about this privacy policy?
Email: privacy@starpage.org
Address: [Your Business Address]

## 8. POLICY UPDATES
We may update this policy. Changes are effective immediately.
Continued use after changes constitutes acceptance.
```

**Action**: Host privacy policy at public URL

- Must be accessible at: `yoursite.com/privacy`
